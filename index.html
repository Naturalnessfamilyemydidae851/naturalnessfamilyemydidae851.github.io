🎯 What is IRIS?

IRIS acts as a smart bridge for your artificial intelligence needs. If you use different AI services, you often deal with separate accounts and confusing settings. IRIS solves this by organizing all your AI tools into one simple system. It manages connections, keeps your data moving, and ensures your work continues even if one service encounters an error.

⚙️ Why Use IRIS?

You gain access to sixteen different AI providers through one interface. You do not need to switch between websites or install separate applications for each service. IRIS handles the complex background tasks so you can focus on your projects. It monitors your tool usage, logs your activity for your own records, and keeps your operations running at high speed.

📥 Setup and Installation

Follow these steps to install IRIS on your Windows computer:

  1. Visit the official download page.
  2. Locate the section labeled Releases or Assets.
  3. Select the file ending in .exe suited for Windows.
  4. Save this file to your computer.
  5. Double-click the downloaded file to start the installation.
  6. Follow the screen prompts until the setup finishes.

🚀 Starting Your First Session

Once installation finishes, an IRIS icon appears on your desktop. Click the icon to open the main window. You will see a command line interface. This interface allows you to add specific AI providers. To add a new provider, type iris provider add followed by the provider name and follow the requests for your credentials. IRIS stores these keys securely on your local device.

🛠️ Managing AI Tools

IRIS supports thirteen distinct tools designed for specific tasks. These tools bridge the gap between your applications and remote AI models. When you add a provider, these tools become available for your use. The system checks for connectivity issues every few seconds. If a connection fails, the system switches to a backup provider automatically to keep your workflow intact.

📈 Monitoring Performance

The system creates audit logs for every command you perform. You can review these logs at any time to see which AI models you used and how long each request took. This helps you track costs and performance. If you want to see detailed usage statistics, IRIS provides a built-in tracking feature that displays how many requests you have sent today.

💻 System Requirements

Your computer needs the following to run IRIS effectively:

  • Windows 10 or Windows 11.
  • At least 4GB of available memory.
  • An active internet connection for model communication.
  • Basic knowledge of typing simple commands.

🛡️ Security and Privacy

IRIS runs locally on your machine. We do not store your private credentials on external servers. Every request you initiate through the IRIS router stays within your control. You manage your own API keys. If you decide to uninstall the software, simply delete the program folder to remove all stored local data.
